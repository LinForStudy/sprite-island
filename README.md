# 萌灵小岛

原创网页小游戏，适合横屏浏览器游玩。核心玩法是探索小岛、发现萌灵、使用星糖罐捕捉、点亮图鉴，并在小屋里喂食、洗澡、摸摸。

## 运行

```bash
npm.cmd install --cache .\.npm-cache
npm.cmd run dev -- --port 5174
```

打开：

```text
http://127.0.0.1:5174
```

## 构建

```bash
npm.cmd run build
```

## 已实现

- 12 只原创 SVG 萌灵。
- 横屏小岛地图，包含草丛、池塘、暖石、风车、山洞、云台、小屋。
- 星糖罐捕捉道具和捕捉动画。
- 图鉴发现/入住状态。
- 小屋喂食、洗澡、摸摸养成。
- 浏览器 `localStorage` 自动存档。
- 竖屏提示横屏游玩。
